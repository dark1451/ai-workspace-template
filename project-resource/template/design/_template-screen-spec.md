## 7. 분위기·배경·일러스트

> 소비자 facing UI는 **§6 토큰만으로 끝내지 않는다.** 백오피스형(flat white + grey) 회피.

### 7.1 배경·분위기 레이어

| 레이어 | 설명 | 구현 힌트 |
|--------|------|-----------|
| (예: sky gradient) | 상단→하단 색 전환 | CSS `linear-gradient` |
| (예: landscape SVG) | 하단 실루엣·패턴 | `public/` SVG, fixed bottom |
| (예: grain overlay) | 종이·질감 | CSS opacity + noise |

### 7.2 히어로 일러스트 / 마스코트

| 항목 | 내용 |
|------|------|
| **형태** | (예: 캐릭터 SVG 72×80px, 상태별 표정) |
| **위치** | (예: 홈 히어로 중앙, 기록 완료 toast 옆) |
| **상태** | idle / active / celebrate / rest — 데이터·달성에 따라 |
| **에셋** | `public/…` 또는 inline SVG 경로 |

### 7.3 아이콘·UI 그래픽

- 스타일: (라인 / 솔리드 / 3D glass)
- empty vs filled: (예: 7일 트레일, 걸은 날 아이콘 채움)
